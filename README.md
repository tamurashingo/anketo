忘年会の景品アンケート
======================

使用技術
--------

- フロントエンド
    - jQuery
- バックエンド
    - Firebase


設定
----

### インストール ###

```
npm install -g firebase-tools
```


### ログイン ###

```
firebase login
```

### デプロイ ###

```
firebase deploy
```
